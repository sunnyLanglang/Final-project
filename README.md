# 自我之路

## 项目简介

《自我之路》是一款帮助中国留学生适应圣彼得堡学习和生活的互动叙事游戏。

游戏模拟了留学生在圣彼得堡从出发到毕业期间可能遇到的真实场景——包括学习压力、生活困难、文化差异、社交挑战等。玩家通过选择不同的应对方式，影响角色的状态，并学习每个问题的最佳做法。通关后，游戏会整理一份完整的建议清单，为未来的留学生活提供参考。

## 游戏特色

- 44个主线事件 + 14个随机事件，覆盖留学全周期
- 支持中俄双语界面，一键切换
- 每个事件三种选择（最佳/中性/最差），影响角色状态
- 累计得分决定三种结局（优秀/普通/艰难）
- 通关后列出所有问题的最佳做法
- 图片和音效反馈，提升沉浸感

## 运行方法

### 环境要求
- Python 3.8 或更高版本
- Pygame 库

安装 Pygame：
pip install pygame

### 启动游戏
在项目目录下打开终端，运行：
python main.py

# Путь к себе

## Описание проекта

«Путь к себе» — это интерактивная игра-нарратив, созданная чтобы помочь китайским студентам адаптироваться к жизни и учёбе в Санкт-Петербурге.

В игре моделируются реальные ситуации, с которыми студенты сталкиваются во время обучения — от прилёта до выпуска: учебные трудности, бытовые проблемы, культурные различия, социальные вызовы. Игрок выбирает варианты поведения, влияя на состояние персонажа, и узнаёт лучшие способы решения каждой проблемы. После прохождения игра выдаёт полный список рекомендаций для реальной жизни.

## Особенности

- 56 основных событий + 14 случайных, охватывающих весь период учёбы
- Поддержка двух языков (китайский/русский), переключение в один клик
- Три варианта выбора (лучший/нейтральный/худший), влияющие на состояние
- Накопительные очки определяют три финала (отличный/обычный/трудный)
- После прохождения — список лучших решений для всех событий
- Визуальные и звуковые эффекты

## Запуск

### Требования
- Python 3.8 или выше
- Pygame

Установка Pygame:
pip install pygame

### Запуск игры
В терминале в папке проекта выполните:
python main.py
